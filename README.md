# All Things Perfume

## 🤔 What is it? 
I formulate and create fragrances from scratch. From raw material selection, to modifications, to a final formula. Currently, my ingredient library is ~200 items long, and it will continue to expand as I build my brand 

## ⁉️ Why does it Matter?
